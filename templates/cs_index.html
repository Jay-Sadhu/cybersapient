About the Project

Welcome to the Movie Review Sentiment Analysis Project, I am Sadhu Jay Vardhan, a student pursuing B.Tech at Mahindra University. This project was undertaken as an assignment for Cybersapient, a renowned company specializing in cutting-edge technologies.

The objective of this project is to provide an intelligent system that performs sentiment analysis on movie reviews, allowing users to gain insights into the overall sentiment expressed in the reviews. The system leverages advanced natural language processing techniques and machine learning algorithms to accurately classify reviews as positive or negative. By analyzing the sentiments, users can make informed decisions and better understand public opinions regarding movies. The project utilizes the popular Flask framework and integrates with a powerful sentiment analysis model trained on the IMDB dataset. The model employs a combination of text preprocessing, TF-IDF vectorization, and logistic regression to achieve robust sentiment classification.
